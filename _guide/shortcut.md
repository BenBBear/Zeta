# Overview

Zeta provides shortcuts for some functions, for more convenient to use.


# Shortcuts

~~~javascript

m.handler  ===> m.h
m.provider ===> m.p
m.factory  ===> m.f
m.config   ===> m.c
m.load     ===> m.l
m.guard    ===> m.g 
m.guard.with ===> m.g.w
~~~
